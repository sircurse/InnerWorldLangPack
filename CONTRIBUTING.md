# Contributing to InnerWorld Resource Pack Languages

Thank you for helping improve the InnerWorld language pack.

This repository accepts contributions for translation files, localization strings, corrections, and language improvements used by InnerWorld / InnerPNX resource packs.

## Contribution Rules

By submitting a pull request, issue, suggestion, translation, correction, or any other contribution to this repository, you agree that:

* Your contribution may be used, modified, distributed, and included in InnerWorld / InnerPNX.
* Your contribution is provided under the same license as this repository.
* Your contribution does not create any right to payment, royalties, revenue share, subscription income, ownership, or control over InnerWorld / InnerPNX.
* InnerWorld / InnerPNX may be used in free or paid services, including subscriptions.

## Translation Guidelines

Please try to keep translations:

* Clear and easy to understand.
* Consistent with the existing language style.
* Close to the original meaning.
* Free from jokes, personal comments, or unrelated text.
* Safe for all players.

## Pull Requests

When opening a pull request, please include:

* The language changed.
* A short description of what was updated.
* Any context needed to understand the translation choice.

Small corrections are welcome.
